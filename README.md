# AMOR_14_FEB

Animación web interactiva en React con partículas cálidas que:

1. Muestra un botón **Iniciar**.
2. Inicia partículas flotando al presionar el botón.
3. Reacciona al movimiento del mouse con una ligera dispersión.
4. Forma secuencialmente los textos: **Te amo**, **mi niña**, **preciosa**, **gracias**, **por existir**.
5. Termina formando un corazón **relleno** con varios corazones internos y animación de pulso.

## Mejoras incluidas

- Botón de audio en la parte superior (icono de bocina), activo por defecto.
- Cambio de paleta de colores brillantes en cada cambio de palabra.
- Partículas ambientales flotando para que el fondo no se vea "limpio".
- Se eliminaron los subtítulos inferiores.

## Importante sobre el audio (sin binarios en PR)

Este repositorio **no incluye** MP3 para evitar errores de plataformas que no aceptan binarios en la PR.

Antes de ejecutar, agrega tu música aquí:

- Archivo: `public/musica.mp3`
- Ya tienes una guía en: `public/musica.example.txt`

## Ejecutar localmente

```bash
npm install
npm run dev
```

Abre el enlace que te muestra Vite (normalmente `http://localhost:5173`).
